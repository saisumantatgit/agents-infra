# Datastore Throughput — Benchmark Summary

In our controlled benchmark on a single node, Redis sustained approximately 128000 operations per second [S1].

Under the same benchmark load and hardware, PostgreSQL sustained approximately 11000 write operations per second [S2].

MongoDB sustained approximately 45000 operations per second in the same test [S3].

Redis delivered about 100 times the throughput of the disk-backed alternative [S1].
